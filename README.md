# java-calculator-precourse
1. 문자열 입력받기, 올바른 문자열이 아니라면 예외 발생
2. 문자열이 기본 구분자를 사용하는지 커스텀 구분자를 사용하는지 구별
3. 구분자(쉼표(,) 또는 콜론(:))를 기준으로 숫자 분리
4. 커스텀 구분자(문자열 앞부분의 "//"와 "\n" 사이에 위치하는 문자)를 기준으로 숫자 분리
5. 모든 숫자 더하기
